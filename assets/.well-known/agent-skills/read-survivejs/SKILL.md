# Read SurviveJS

Use this skill when answering questions from SurviveJS content.

## Source Selection

- Start with `https://survivejs.com/llms.txt` for the site map.
- Use `https://survivejs.com/sitemap.xml` when you need exhaustive page discovery.
- Prefer book pages for stable conceptual explanations.
- Prefer blog pages for interviews, announcements, and historical context.
- Include publication or update dates when comparing current and older material.

## Answering Guidance

- Cite canonical SurviveJS URLs.
- Preserve technical nuance from the surrounding section or chapter.
- Mention when a post is historical and may describe older versions of a tool.
- Do not imply that book material is fully current unless the page itself indicates that.
